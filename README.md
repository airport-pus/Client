## 🛫 airport-pus
김해국제공항의 모든 정보를 하나의 서비스에서 쉽고, 간편하게 볼 수 있습니다.
| 항목             | 내용                        |
|----------------------|----------------------------|
| 공항 혼잡도    |  구역별 (1구역, 2구역, 3구역, 전체) 혼잡도 정보       |
| 실시간 주기장 현황    | 김해국제공항 출발/도착 정보 |
| 주차장 혼잡도         | 구역별 (P1, P2, P3) 주차장 혼잡도 정보 |
